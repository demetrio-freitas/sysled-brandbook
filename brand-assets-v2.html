08 — Do's & Don'ts

Uso dos Brand Assets

Regras fundamentais para uso correto dos assets da marca SYSLED.

✓ Faca

Sempre use os arquivos SVG oficiais do kit — nunca recrie o logo manualmente

✕ Nao Faca

Nunca altere cores, proporcoes ou elementos do logotipo

✓ Faca

Use a versao correta do logo para cada tipo de fundo (claro, escuro, colorido)

✕ Nao Faca

Nunca use a logo colorida sobre fundos escuros ou a branca sobre fundos claros

✓ Faca

Mantenha a area de protecao ao redor do logo (min. 50% da altura da nuvem)

✕ Nao Faca

Nunca coloque elementos visuais invadindo a area de protecao do logo

✓ Faca

Use apenas Montserrat, Inter e JetBrains Mono — as fontes oficiais da marca

✕ Nao Faca

Nunca substitua as fontes por alternativas — mesmo que visualmente parecidas

09 — Checklist

Checklist de Implementacao

Verifique cada item ao implementar a identidade visual SYSLED em qualquer projeto ou material.

Logos SVG oficiais copiados para o projeto (4 variantes)
Google Fonts importadas (Montserrat + Inter + JetBrains Mono)
CSS Design Tokens configurados (:root com custom properties)
Paleta de cores aplicada conforme o brandbook
Gradientes de marca utilizados corretamente (brand, accent, dark)
Favicon configurado com sysled-icone.svg
Dark mode implementado com [data-theme="dark"] e tokens corretos
Acessibilidade verificada (contraste WCAG AA, alt texts, aria-labels)
Responsividade testada em mobile, tablet e desktop
OG Image (1200x630) com identidade visual para compartilhamento
