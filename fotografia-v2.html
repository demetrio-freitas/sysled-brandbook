Fotografia & Imagens

Diretrizes para estilo fotografico, tratamento de cor, enquadramento, overlays de marca e banco de imagens da SYSLED.

01 — Principios

Fundamentos Visuais

A fotografia da SYSLED transmite tecnologia, confianca e modernidade industrial. Cada imagem deve reforcar o posicionamento da marca como plataforma indtech inteligente.

Clareza
Imagens limpas, bem iluminadas e com foco definido. Sem ruido visual ou elementos que distraiam.
Profundidade
Composicoes com camadas e perspectiva. Transmitir a complexidade da industria com elegancia.
Autenticidade
Preferir fotos reais do ambiente industrial. Evitar imagens genericas de banco sem contexto.
Conectividade
Mostrar conexao entre sistemas, maquinas e dados. A nuvem como elo central.
Luminosidade
Tom geral frio com toques de luz quente. Contrastar tecnologia (azul) com humanidade (dourado).
Escala
Valorizar a grandeza da operacao industrial. Perspectivas amplas e detalhes macro.
02 — Estilos Fotograficos

4 Categorias de Imagem

Toda imagem usada em materiais da SYSLED deve se encaixar em uma destas 4 categorias. Cada uma tem tratamento de cor e proposito especifico.

Industrial / Chao de Fabrica
Maquinas, linhas de producao, galpoes, equipamentos. Tom frio com dominante azul-acinzentado.
Uso Primario
Tecnologia / Dashboards
Telas, graficos, interfaces, dados. Fundo escuro com brilhos azuis e pontos de luz.
Digital
Pessoas / Equipe
Colaboradores em acao, engenheiros, gestores. Luz natural, foco no rosto, ambiente profissional.
Institucional
Dados / Abstrato
Visualizacoes de dados, redes, fluxos. Para backgrounds e hero sections. Gradientes da paleta SYSLED.
Background
03 — Tratamento de Cor

Filtros e Color Grading

Todas as fotos devem receber tratamento de cor para manter consistencia visual. O tom geral e frio (azulado), com saturacao moderada.

Filtro Padrao SYSLED

/* Filtro padrao para fotos SYSLED */
.photo-sysled {
  filter:
    saturate(0.85)      /* Reducao leve de saturacao */
    contrast(1.05)      /* Contraste sutil */
    brightness(0.95);   /* Levemente mais escura */
  mix-blend-mode: normal;
}

/* Overlay azul opcional */
.photo-sysled::after {
  content: '';
  position: absolute;
  inset: 0;
  background: linear-gradient(
    135deg,
    rgba(15, 23, 42, 0.3),   /* #0F172A */
    rgba(43, 108, 176, 0.15)  /* #2B6CB0 */
  );
  mix-blend-mode: multiply;
}

04 — Overlays de Marca

Gradientes sobre Fotos

Quando a foto precisa de texto sobreposto, use um overlay gradiente da paleta SYSLED. Nunca coloque texto diretamente sobre fotos sem tratamento.

Dark Bottomlinear-gradient(180deg, transparent 30%, #0F172A 100%)
Brand Gradientgradient-start → gradient-end @ 0.8
Dark Uniformrgba(15, 23, 42, 0.6)
/* Overlay tipo 1: Dark bottom (ideal para cards com texto inferior) */
.overlay-dark-bottom::after {
  background: linear-gradient(180deg, transparent 30%, rgba(15,23,42,0.85) 100%);
}

/* Overlay tipo 2: Brand gradient (ideal para heroes e covers) */
.overlay-brand::after {
  background: linear-gradient(135deg, rgba(26,78,138,0.85), rgba(26,122,143,0.7));
}

/* Overlay tipo 3: Uniform dark (ideal para texto centralizado) */
.overlay-uniform::after {
  background: rgba(15, 23, 42, 0.6);
}

08 — Do's & Don'ts

Uso Correto vs Incorreto

Exemplos praticos do que fazer e do que evitar na fotografia da marca.

Correto

Tom frio, filtro SYSLED aplicado, contraste equilibrado

Incorreto

Tom quente/alaranjado — conflita com a identidade azul da marca

Correto
Titulo sobre foto

Texto com overlay escuro — legibilidade garantida

Incorreto
Titulo sem overlay

Texto direto sobre foto — baixo contraste, ilegivel

Correto

Fotos reais de industria, maquinas, tecnologia

Incorreto
🏖️

Imagens genericas sem relacao com industria/tecnologia

09 — Checklist

Validacao de Imagem

Antes de usar qualquer imagem em materiais SYSLED, valide com este checklist:

A imagem transmite tecnologia, industria ou profissionalismo?
O filtro SYSLED (saturate 0.85, contrast 1.05, brightness 0.95) foi aplicado?
A resolucao e suficiente para o contexto (min. 72dpi digital, 300dpi impresso)?
Nao contem logos de terceiros, marcas d'agua ou conteudo inadequado?
Se ha texto sobreposto, existe overlay com contraste suficiente?
A licenca da imagem permite uso comercial?
A proporcao esta correta para o contexto (16:9, 4:3, 1:1, etc.)?
